Smart Secure Motion Lock

EEC 172 Final Project

Zijian Li – UC Davis

Project Overview

This project implements a smart embedded security system using the TI CC3200 LaunchPad. The system monitors motion using an accelerometer and allows the user to arm or disarm the system through an IR remote password. When suspicious motion is detected, the system sends an alert message to AWS IoT which can trigger an email notification.

System Architecture

The CC3200 acts as the main controller. The accelerometer communicates through I2C to provide motion data. The IR receiver captures user input, and the WiFi interface sends alert messages to AWS IoT when motion is detected.

Reproducibility

To recreate this project, a student would need a CC3200 LaunchPad, an accelerometer module, an IR receiver, and an AWS IoT configuration. The software runs in Code Composer Studio and communicates with AWS through HTTPS.

Challenges

Integrating multiple subsystems such as motion sensing, IR decoding, and cloud communication required careful debugging. Selecting an appropriate motion threshold was also necessary to avoid false alerts.
